# 🌍 Ajuda.AI 
**Orquestrando o voluntariado inteligente em situações de crise**

---

## 📌 Visão Geral

O **Ajuda.AI** é uma plataforma web que conecta voluntários a projetos sociais e situações de emergência de forma inteligente, utilizando **Inteligência Artificial e automação**.

Nosso objetivo é resolver um problema crítico:  
> pessoas querem ajudar, mas não sabem **como**, **onde** e **com qual impacto real**.

A plataforma utiliza um agente de IA baseado no **IBM watsonx Orchestrate** para entender o perfil do usuário e recomendar ações personalizadas, maximizando o impacto social.

---

## 🚨 Problema

Em cenários de vulnerabilidade social, desastres naturais e ações comunitárias:

- Voluntários não sabem como aplicar suas habilidades  
- Instituições não conseguem organizar ajuda de forma eficiente  
- Falta priorização e inteligência na distribuição de esforços  
- Existe pouca utilização de dados para tomada de decisão  

Resultado:  
⚠️ Desperdício de potencial humano  
⚠️ Baixo impacto das ações  
⚠️ Respostas lentas em momentos críticos  

---

## 💡 Solução

O Ajuda.AI propõe uma plataforma baseada em:

### 🔹 Mapeamento inteligente de voluntários
- Habilidades  
- Disponibilidade  
- Localização  
- Preferências  

### 🔹 Cadastro de demandas sociais
- Tipo de necessidade  
- Nível de urgência  
- Número de voluntários necessários  

### 🔹 Recomendação com IA
Um agente de IA conduz uma conversa com o usuário para:
- Entender seu perfil  
- Identificar suas afinidades  
- Sugerir projetos ideais  

### 🔹 Orquestração de ações
- Conexão entre voluntários e instituições  
- Fluxo organizado de execução  
- Acompanhamento das atividades  

---

## 🧠 Uso de Inteligência Artificial

A solução utiliza o **IBM watsonx Orchestrate** como núcleo de inteligência para:

- Coletar informações do usuário via agente conversacional  
- Interpretar habilidades e contexto  
- Sugerir ações de impacto alinhadas ao perfil  
- Apoiar na tomada de decisão em cenários de crise  

---

## ⚙️ Arquitetura do Sistema

A arquitetura foi projetada para ser simples, escalável e eficiente dentro do contexto de um hackathon.

### 🔧 Backend
- Node.js  
- API responsável por:
  - Gerenciar usuários  
  - Gerenciar demandas  
  - Integrar com o agente de IA  
  - Processar recomendações  

### 🎨 Frontend
- Interface web simples e intuitiva  
- Foco em experiência do usuário  

### 🗄️ Banco de Dados
- Banco local (durante o desenvolvimento)  
- Armazenamento de:
  - Usuários  
  - Habilidades  
  - Demandas  
  - Recomendações  

### 🤖 Integração com IA
- Comunicação via API com o agente criado no watsonx Orchestrate  
- O agente atua como camada de inteligência e recomendação  

---

## 🔁 Fluxo do Usuário

1. Usuário acessa a plataforma  
2. Interage com o agente de IA  
3. Responde perguntas sobre:
   - Habilidades  
   - Disponibilidade  
   - Interesses  
4. Sistema processa as informações  
5. Plataforma sugere:
   - Projetos existentes  
   - Comunidades  
   - Possíveis ações  
6. Usuário escolhe como deseja contribuir  

---

## 🎯 Diferenciais

- Uso real de IA na recomendação  
- Foco em **impacto social mensurável**  
- Orquestração inteligente de voluntariado  
- Experiência guiada (reduz fricção do usuário)  
- Escalável para cenários de crise  

---

## 📊 Estratégia de Recomendação

A recomendação é baseada em:

- Compatibilidade de habilidades  
- Nível de urgência da demanda  
- Disponibilidade do usuário  
- Contexto da situação  

O objetivo é garantir que:
> a pessoa certa ajude na tarefa certa, no momento certo
